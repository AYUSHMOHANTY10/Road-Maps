# next.roadmap.sh
WIP version of roadmap.sh built with 11ty and Tailwind.

## Development

Clone the repo and install dependencies:

```bash
git clone git@github.com:kamranahmedse/roadmap.sh.git
pnpm install
```

Run the project

```shell
pnpm run dev
```
